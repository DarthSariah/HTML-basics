CSS - Cascading Style Sheets

To apply CSS we have 3 ways

  • Inline Styles
  • Internal Styles
  • External Styles

Inline styles are styles we add directly over an element inside the HTML page using the attribute style.

Internal styles are styles we put inside the head element of our HTML document using the style element tag.

External styles are styles we write in a file with .css extention and link it using the link element tag inside our head tag in the HTML document. The link tag takes 2 attributes, rel (which needs the value 'stylesheet') and the src which needs the file name.

Notes

Using styles we got a hierarchy. What that means is that inline styles override the other two (external, internal).

Internal styles override the external ones.

After every CSS rule, we need to end with a semicolon (;)
